my blog guide

step 1 - obsidian

write the post in obsidian on my mac, and once done, export to html with plugin and move the file to /blog/

step 2 - formatting

with the exported html file, there will be little formatting done to fit the webpage. Import the base css formatting file with @import url("../base.css");

step 3 - indexing

now that the scroll bar is formatted correctly, import the page into my blog index. To do this, basically just copy the previous entry and replace date modified, size, and the link to the file. The base64 png should stay static.

step 4 - done
